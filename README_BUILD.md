# Как собрать мод онлайн через GitHub

1. Создайте новый репозиторий на GitHub (например, `my-drawing-mod`).
2. Загрузите в него **все** файлы из этого архива.
3. Перейдите на вкладку **Actions** в вашем репозитории.
4. Вы увидите запущенный процесс сборки (workflow "Java CI with Gradle").
5. Дождитесь окончания (зеленая галочка).
6. Нажмите на завершенную сборку, прокрутите вниз до раздела **Artifacts**.
7. Скачайте файл `DrawingMod-Jar`. Внутри него будет ваш готовый `.jar` мод!
